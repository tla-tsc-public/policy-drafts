# TSC Policy Drafts

Working drafts developed by the Technical Steering Committee for 
interoperable learning data infrastructure.

## Current Drafts

### 1. Open Source Policy
Guidance on preferred and acceptable open source licenses, 
stewardship and sustainability expectations, transparency 
requirements, and commitments to equity and inclusivity in 
open source development.

### 2. xAPI LRS Test Suite Advisory Plan
Recommendations on what the xAPI LRS Test Suite needs to be 
and what it needs to do, including automated conformance testing, 
certification policy, and governance of the test suite as a 
shared community resource.
